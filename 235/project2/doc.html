Information

  • Can search for any string of characters
  • Search by either 'Country' or 'Capital'
  • Sort by A -> Z or Z -> A
  • Displays flags and country/capital name

Features

  • Stores all info locally
  • When a filter is changed, the search is updated
  • Bold changes to highlight whichever type was searched
  • Page responds well to different widths and heights
  • Used feedback given in class to make the website better
